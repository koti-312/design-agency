# Nova Agency - Design Agency Homepage

A modern Design Agency Homepage built with Next.js.

---

## Live Demo

Deployed on vercel - https://design-agency-ebon.vercel.app

---

## Tech Stack
- HTML
- CSS
- Javascript
- Next.js version 16
 
---

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/design-agency.git
```

2. Install dependencies:
```bash
npm install
```

3. Run development server:
```bash
npm run dev
```

4. Open in browser:

http://localhost:3000
